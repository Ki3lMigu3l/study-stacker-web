Sobre o projeto

Este projeto em andamento visa consolidar os conhecimentos adquiridos nos cursos de formação da Alura, especialmente na formação "Desenvolva Aplicações Web com JavaScript". Utilizando tecnologias fundamentais da web, como HTML, CSS e JavaScript, a iniciativa busca explorar recursos avançados para proporcionar uma experiência interativa e inovadora.

Recursos Principais:

Reconhecimento de Voz (Web Speech API): Implementamos a Web Speech API para integrar funcionalidades de reconhecimento de voz à aplicação. Essa tecnologia possibilita comandos de voz, oferecendo uma experiência única e acessível aos usuários.

Carrossel Dinâmico (Swiper JS): Incorporamos a biblioteca Swiper JS para criar um carrossel dinâmico e responsivo. Essa funcionalidade não apenas melhora a navegação, mas também proporciona uma apresentação visualmente atraente do conteúdo.

Objetivo Final:

O projeto em andamento servirá como documentação prática para os conceitos e técnicas abordados na formação "Desenvolva Aplicações Web com JavaScript" da Alura. Além de consolidar o aprendizado, a aplicação pretende ser uma referência para futuros projetos, demonstrando a aplicação efetiva dos conhecimentos adquiridos.

Este é um passo importante na jornada de aprendizado, explorando as fronteiras da interatividade e usabilidade na web moderna. Acompanhe-nos enquanto continuamos a desenvolver e aprimorar esta aplicação, explorando novas possibilidades e integrando mais tecnologias avançadas.
